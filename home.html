Info about asteroids

The Near Earth Object Web Service is a RESTful web service for near earth Asteroid information. Here it has been used to search for Asteroids closest to Earth. It is also possible to browse the overall data-set and in this case name, speed, and size were gathered for every object considered potentially dangerous. Data-set: All the data is from the NASA JPL Asteroid team http://neo.jpl.nasa.gov/.
